# fx-auto-trade
FXの自動売買ツール(開発途中)

## 仮想環境構築手順
1. git clone
$ git clone https://github.com/regnan/fx-auto-trade

2. virtualenv構築
$ python -m venv venv

3. 仮想環境をアクティベート
$ \venv\Scripts\activate.bat

4. requirement.txtからpipでパッケージを入れる
$ pip install -r requirements.txt

5. TA-Libをインストール
https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib

6. 同期完了
$ pip list
